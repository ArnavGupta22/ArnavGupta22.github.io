Discovery Project

Google Docs Compatible Voice-Activated Task Manager

Required Technology

RaspberryPi, MAX4466, ADS1115, Micro SD Card, Breadboard, Jumper Wires

Project Description

Have you ever suddenly remembered an important task while in bed, but didn't want to get up to write it down? Yeah, me too. I created a voice-activated task manager that allows you to add tasks to a Google Docs using voice commands. You speak your task into the microphone, the RaspberryPi processes the audio, and the task is added to your Google Docs.

Steps Taken

1. Set up RaspberryPi: Configure settings and note IP address

2. Hardware Setup: Wire and setup the MAX4466 and ADS1115 with the RaspberryPi

3. Google Docs API: Enable API and obtain credentials.json file

4. Python Script: Write a Python script to process audio and add tasks to Google Docs

5. Run the Program!

Challenges

1. Learning how to use the API to add tasks to a Google Docs

2. Wiring the MAX4466 and ADS1115 to the RaspberryPi. No access to USB microphone, requiring an analog to digital converter.

3. Debugging the Python script to ensure the audio was being processed correctly and being added to the Google Doc.
